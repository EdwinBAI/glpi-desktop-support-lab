# GLPI Desktop Support Lab

Hands-on Desktop Support lab using GLPI, focused on real-world help desk workflows.

## Scope
This lab demonstrates Level I / Level II desktop support skills commonly required in healthcare IT environments.

## Environment
- GLPI (Docker-based deployment)
- Windows 10/11 clients
- Simulated users and incidents
- Ticket lifecycle: intake → triage → resolution → verification → closure

## Completed Scenarios
- Outlook login and profile issues
- Printer queue and spooler failures
- Windows login and MFA problems
- Network connectivity troubleshooting
- Basic hardware and performance issues

## Artifacts Included
- Sample GLPI tickets with documented troubleshooting steps
- Knowledge Base (KB) article for common printer issues
- Screenshots showing ticket creation, resolution, and closure

## Skills Demonstrated
- Windows desktop troubleshooting
- Outlook and Office support
- Printer and spooler management
- Login, password reset, and MFA support
- ITSM documentation and ticket hygiene
- Clear communication and escalation judgment

## Notes
This repository is intended as a skills demonstration for desktop support and help desk roles.
